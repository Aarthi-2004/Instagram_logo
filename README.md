# Instagram_logo
Instagram Logo using HTML & CSS:

This project recreates the Instagram logo using pure HTML and CSS without using any images. It demonstrates how different CSS properties such as gradients, borders, border-radius, and positioning can be used to design a recognizable UI component.

Overview
The Instagram logo is built using simple HTML structure and advanced CSS styling techniques. The design includes a gradient background, camera outline, lens, and flash indicator, all created with CSS.

Features
Built completely using HTML and CSS
No images or external graphics used
Uses CSS gradients to replicate Instagram colors
Clean and minimal code structure
Responsive centered layout using Flexbox

Technologies Used
HTML5
CSS3

How It Works

The outer container centers the logo on the page using Flexbox.
A radial gradient background recreates the Instagram color theme.
The camera outline is created using border and border-radius.
The inner lens is designed using nested elements.
A CSS pseudo-element (::after) creates the small flash circle in the top corner.
