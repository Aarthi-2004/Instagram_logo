# Instagram_logo

📸 Instagram Logo – HTML & CSS Project


📌 Project Overview


The Instagram Logo Project is a simple front-end design project that recreates the popular Instagram logo using only HTML and CSS, without using any images or graphic files.


This project demonstrates how CSS styling techniques such as gradients, borders, border-radius, flexbox, and pseudo-elements can be used to design complex UI shapes.


The design closely resembles the original Instagram logo and is displayed in the center of the screen with a clean and modern layout.



🎯 Objective of the Project


The main objective of this project is to:




Practice CSS layout and styling techniques




Understand how to create shapes using CSS borders and border-radius




Learn how to use CSS gradients




Improve front-end design skills




Understand the use of pseudo-elements in CSS





🛠️ Technologies Used




HTML5 – Structure of the web page




CSS3 – Styling and design of the Instagram logo

🧠 How the Application Works


Step 1: Page Layout


The webpage contains a container that centers the Instagram logo in the middle of the screen using Flexbox.


Step 2: Logo Background


The outer logo box is created using CSS radial gradient colors to replicate Instagram’s iconic color scheme.


Example gradient colors used:



#fed373 → #f15245 → #d92e7f → #9b36b7 → #515ecf



Step 3: Camera Outline


The camera shape is created using:




Border




Border-radius




Nested div elements




This forms the square camera frame inside the logo.


Step 4: Camera Lens


A circular lens is created inside the frame using border-radius: 50% with a thick border.


Step 5: Flash Indicator


The small circle in the top corner of the camera is created using a CSS pseudo-element (::after).


This helps simulate the camera flash dot seen in the Instagram logo.
